01 ── ABOUT THE JAM

What is Geek n Jam?

A month-long pan-India game development jam organized by Geekroom SRMIST. Solo devs, small teams, artists, designers — build something unforgettable from scratch, then show it off at SRMIST.

◈ THEME: INVERTED ◈

Mechanics, narratives, gravity — flip something expected on its head.

02 ── WHAT TO SUBMIT

Submission Requirements

  • ▶Playable game build — your game must actually run
  • ▶Gameplay video (1–2 minutes) showing core mechanics
  • ▶Source code repository (public GitHub/GitLab)
  • ▶Game description & controls — short and clear
  • ▶Asset credits — list everything you didn't make yourself
03 ── THE RULES

What's Allowed

✕No AI-generated assets
✕No plagiarism
✓Paid assets (must declare)
✕No pre-built games
✓All engines welcome
✓Solo or team play

⚠ Game must be built entirely during the jam period. Previously built games will not be considered.
